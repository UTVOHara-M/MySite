
## Как запустить

1. Убедись, что **Laragon** установлен и запущен.
2. Нажми **Start All** 

## База данных

- Название базы: `mysite`
- Таблица: `messages`
- Поля:
  - `id` — автоинкремент
  - `name` — имя отправителя
  - `email` — email
  - `message` — текст сообщения
  - `created_at` — дата и время отправки

Просмотреть данные можно через phpMyAdmin:  
Laragon → **Database** → Open
